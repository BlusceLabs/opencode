ชุดโครงการชุมชนที่สร้างขึ้นบน ClawC

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ ClawC ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-clawc](https://github.com/awesome-clawc/awesome-clawc) และ [clawc.cafe](https://clawc.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
